INTRODUCTION
------------


Gift Certificates for Drupal Commerce.

This module doesn't provide fully-fledged support for gift certificates;
rather, it supplies certain elements of this functionality.
For details on what it does, how it works and what else you need to do to get gift certificates set up in Drupal Commerce, visit this blog post:

http://lukaswhite.com/blog/post/2013/gift-certificates-drupal-commerce
